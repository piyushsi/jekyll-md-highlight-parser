# jekyll-md-highlight-parser

A [Jekyll](https://jekyllrb.com/) Markdown Highlight parser.

## Installation

```
$ npm install jekyll-md-highlight-parser
```

## Usage

```javascript
import { mdParse } from "jekyll-md-highlight-parser";
mdParse(string);
```
